# The Wash

The sequencing run had finished.

Data was already moving toward analysis. The cartridge was still in the machine. A washing cycle came next, the same way it always did. The procedure was familiar enough that nobody needed instructions anymore. The liquids were prepared. The cartridge was opened. Small volumes went where they were supposed to go.

There were a lot of places they were supposed to go.

One compartment filled. Then another. Then another. The process repeated across the cartridge until every slot contained what it needed. Nothing about the task was difficult. Nothing about it required special judgment. It simply took attention and a few minutes of work before the machine could continue through its cleaning cycle.

The sequence completed without trouble.

A person encountering the process for the first time often stopped and looked at the cartridge. The reaction was predictable. There were many openings. The same liquid was distributed repeatedly. The question arrived almost immediately: why was this still being done by hand?

The wash still happened.

A bottle was retrieved. Liquid was measured. Compartments were filled. The machine accepted the cartridge and continued. The procedure worked exactly as designed.

The task stayed in the workflow because it had always been there.

Over time, the irritation faded. Familiarity made the repetition smaller. Five minutes of preparation no longer felt like much. The machine produced reliable sequencing data. Runs completed successfully. Results arrived. The washing cycle became one more thing that happened between jobs.

Something else happened alongside it.

The machine itself earned a reputation. New runs were trusted. Libraries went in. Data came out. The touchscreen could feel sluggish. The footprint occupied more space than anyone would prefer. Supplies arrived wrapped in layers of packaging that eventually filled disposal bins. None of those things disappeared. They accumulated quietly around the edges of the work.

The cartridge still had to be filled.

Yet the task no longer carried the same weight as the first time someone encountered it. The repetition had acquired context. The person standing in front of the machine was not filling compartments for a machine that had failed to help. They were preparing one that had already done its job.

By the end of the cycle, the cartridge was back in place and the machine continued through its cleaning process. The sequence closed the same way it always had.

A few days later another run finished, and the cartridge appeared again, waiting for the same liquid in the same openings, attached to a machine that people trusted enough to keep taking care of.

## What Changed

**Before:** Filling the washing cartridge looked like unnecessary manual work that should have been automated.

**After:** The same task became easier to tolerate because it followed a machine that consistently delivered usable data.

**Now the reader notices:** The cartridge is still cumbersome, but it is attached to something people trust.
